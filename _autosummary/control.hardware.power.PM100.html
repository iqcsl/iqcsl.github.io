class control.hardware.power.PM100(name=None)[source]

Bases: object

Thorlabs PM100 class

Example:

from control.hardware.power import PM100

print(list_resources())
pm = PM100() # connect to first visible resource by default
pm.wavelength = 450 # blue light

print(pm.measure_builtin(1000))
Parameters:

name (Optional[str]) – name of resource (can be found using list_resources()). If not provided, attempts to connect to the first resource found
