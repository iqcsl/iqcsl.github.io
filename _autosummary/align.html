alignÔÉÅ

Script for aligning & calibrating optical setups

Provides a command-line interface which lists different operations/routines, e.x. cross two polarizers, find fast-axis of waveplate, etc.

Additionally, some common alignment routines have been bundled into one command which sets up and does everything in the correct order.

Also allows you to drop into a REPL for manual operation
