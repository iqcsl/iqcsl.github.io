align.Data

class align.Data(X, scatter=(), plot=(), theta=None)[source]

Bases: object

Class for holding measurements & related data

Parameters:

  • X (List[float]) – X-values (typically angles in degrees)

  • scatter (Tuple[List[float]]) – tuple of lists of Y-values to be drawn with plt.scatter() (typically Y-values, e.x. intensity in watts)

  • plot (Tuple[List[float]]) – tuple of lists of Y-values to be drawn with plt.plot() (typically fit data)

  • theta (Optional[float]) – usually an angle which minimizes/maximizes y

Methods

graph

Graph the data and show it using matplotlib

print

Print all the data to the terminal and offers to show a graph
