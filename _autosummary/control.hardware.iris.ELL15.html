control.hardware.iris.ELL15

class control.hardware.iris.ELL15(sn)[source]

Bases: object

Thorlabs ELL15 Iris class

Typically connected to an Interface Board, which is connected to a computer

Example:

from control.hardware.iris import ELL15

iris = ELL15('DK0BJ258') # replace with interface board serial number
print(iris.info())
print(iris.status())
print(iris.max_aperture)
print(iris.min_aperture)
iris.home()
iris.goto(5)
iris.moverel(-0.5)
print(iris.position)

goto(mm, addr=0, blocking=True)[source]

Go to a certain position

The given position must be in the interval [ELL15.min_aperture, ELL15.max_aperture].

Parameters:

  • mm (float) – target position in millimeters

  • addr (int) – address

  • blocking (bool) – if True, wait for move to finish stopping before continuing execution

moverel(mm, addr=0, blocking=True)[source]

Move relatively by a certain amount

The given amount must not move the iris out of the interval [ELL.min_aperture, ELL15.max_aperture].

Parameters:

  • mm (float) – amount to move in millimeters. Positive means increase aperture, negative means decrease aperture

  • addr (int) – address

  • blocking (bool) – if True, wait for move to finish stopping before continuing execution
