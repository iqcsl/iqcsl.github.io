Hardware settings template

A template is a str -> Setting dictionary

e.x. motor serial number, what direction corresponds to clockwise, etc. can be defined as ``Setting``s and placed inside this template

interface = None

The hardware/IO interface provided to the experiment

This exists so that an experiment can operate on different hardware-configurations without having to be rewritten for each one. For example, one setup may use an SLM to generate a vortex pattern and obstruction, while another might use a q-plate along with a motorized iris.
