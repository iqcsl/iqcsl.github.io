showPattern(pattern)[source]

Display a pattern on the SLM

Every pixel on the image is associated with an (x,y) coordinate. These coordinates are passed to the provided function to produce the pattern.

For example:

from control.hardware.slm import GAEA2
from control.hardware.pattern import vortex

slm = GAEA2()
pat = lambda x, y: vortex(x, y, OAM=5)
slm.showPattern(pat)

will display a vortex pattern with OAM 5 on the SLM

Parameters:

pattern (function consuming 2 ints and producing a gray value (integer from 0 to 255)) – the function which produces the pattern
